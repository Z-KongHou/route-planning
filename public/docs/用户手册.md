# 智能路线规划系统 - 用户手册

## 1. 系统介绍

智能路线规划系统是一个基于高德地图 API 开发的纯前端路线规划工具，支持驾车、步行、公交三种交通方式，用户可以输入起点和终点，系统会自动计算最优路线，并支持路线收藏、编辑、删除以及本地持久化。

## 2. 快速开始

### 2.1 环境要求

- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 建议使用最新版本的浏览器以获得最佳体验
- 不需要安装任何软件，直接通过浏览器访问即可

### 2.2 访问系统

1. 启动开发服务器：

   ```bash
   pnpm dev
   ```

2. 在浏览器中访问：
   ```
   http://localhost:5173
   ```

## 3. 功能使用指南

### 3.1 路线规划

1. **进入首页**：登录成功后，系统自动跳转至首页，显示地图、起点终点输入框和交通方式选择。

2. **输入起点和终点**：

   - 在“起点”输入框中输入起点地址，例如：“北京市朝阳区建国路 88 号”
   - 在“终点”输入框中输入终点地址，例如：“北京市海淀区中关村大街 1 号”
   - 系统会自动进行地址解析和补全

3. **选择交通方式**：

   - 点击交通方式选项卡，选择“驾车”、“步行”或“公交”
   - 默认选择“驾车”

4. **搜索路线**：
   - 点击“搜索路线”按钮
   - 系统会调用高德地图 API 计算最优路线
   - 地图上会显示路线，右侧会显示路线详情（距离、时间、关键路段）

### 3.2 AI 聊天功能

1. **使用 AI 助手**：

   - 在首页或其他页面，找到 AI 聊天窗口
   - 在输入框中输入您的问题，例如：“如何规划从北京到上海的路线？”
   - 系统会自动回复相关信息
   - 支持多种路线规划相关问题的咨询

2. **AI 功能特点**：
   - 提供路线规划建议
   - 解答交通方式选择问题
   - 提供地图使用技巧
   - 支持自然语言交互

### 3.3 路线收藏

1. **收藏当前路线**：

   - 在路线规划结果页面，点击“收藏路线”按钮
   - 系统会弹出对话框，要求输入路线名称
   - 输入名称后，点击“确定”按钮
   - 收藏成功后，系统会显示提示信息

2. **查看收藏列表**：

   - 点击页面顶部导航栏的“收藏”按钮
   - 进入收藏页面，显示所有收藏的路线
   - 每条路线显示名称、起点、终点、交通方式和创建时间

3. **编辑收藏路线**：

   - 在收藏页面，找到要编辑的路线
   - 点击路线右侧的“编辑”按钮
   - 系统会弹出对话框，显示当前路线名称
   - 修改名称后，点击“确定”按钮
   - 编辑成功后，系统会显示提示信息

4. **删除收藏路线**：
   - 在收藏页面，找到要删除的路线
   - 点击路线右侧的“删除”按钮
   - 系统会弹出确认对话框
   - 点击“确定”按钮
   - 删除成功后，系统会显示提示信息

### 3.3 路线查看

1. **查看收藏的路线**：

   - 在收藏页面，找到要查看的路线
   - 点击路线名称或“查看”按钮
   - 系统会跳转到首页，并在地图上显示该路线
   - 右侧会显示路线详情

2. **重新计算路线**：
   - 在首页，修改起点或终点
   - 点击“搜索路线”按钮
   - 系统会重新计算路线

## 4. 常见问题

### 4.1 路线计算失败怎么办？

- 检查输入的起点和终点是否正确
- 检查网络连接是否正常
- 尝试刷新页面后重新搜索
- 如问题持续存在，请联系管理员

### 4.2 收藏的路线丢失了怎么办？

- 收藏的路线存储在浏览器的 localStorage 中
- 请不要清除浏览器缓存或使用隐私模式
- 如果清除了缓存，收藏的路线将无法恢复

### 4.3 地图加载缓慢怎么办？

- 检查网络连接是否正常
- 尝试刷新页面
- 关闭浏览器中的其他标签页，释放资源

### 4.4 支持哪些交通方式？

- 驾车
- 步行
- 公交

### 4.5 可以在手机上使用吗？

- 支持在手机浏览器上使用
- 建议使用横屏模式以获得更好的体验
- 支持 iOS 和 Android 系统的主流浏览器
